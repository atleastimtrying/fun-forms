Fun with forms

What are forms and what do they do??

  • Forms are the old school way of communicating between client and server
  • They package up data into key value pairs and send them to a specific url.
  • The use HTTP verbs, normally get or post
  • They used to be the main form of interactivity online.

The advent of AJAX

  • The concept of AJAX came along and changed this idea.
  • Through javascript developers can now send data without any markup at all.
  • This is exactly how sites with real time data like facebook work.
  • They do still use form inputs though.

Form inputs - the old

  • Text

  • Radio Buttons
  • Checkboxes
  • Textareas

Is there a place for form tags?

  • Yes!
  • Having a submit event lets us group key values easily
  • Server side form helpers can give us dynamic paths via the action attribute
  • The method attribute lets us use the right http verb for sending.
  • Falls back nicely for older browsers.
